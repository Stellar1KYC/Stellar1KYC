# Frontend Architecture

## Overview

The StellaroneKYC frontend is built with Next.js App Router, TypeScript, and Tailwind CSS.

## Key Layers

- **app/** — Next.js routes and pages
- **components/** — Reusable UI components grouped by domain
- **services/** — API and blockchain integrations
- **hooks/** — Custom React hooks for shared logic
- **context/** — Global state via React Context
- **store/** — Lightweight global state management
- **types/** — Shared TypeScript interfaces and types
